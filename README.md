## 如何提交你的代码？

我们采用 **Pull Request（PR）** 的方式收集每位成员的项目代码。

### 什么是 PR（Pull Request）？

PR 是 GitHub 上请求将你的代码**合并到主仓库**的标准流程。它不仅是提交方式，也是代码协作与评审的起点。

### ✅ 提交流程（三步走）

1. **Fork 本仓库**  
   点击右上角 **`Fork`** 按钮，将 `training_camp_code_driver` 复制到你的个人 GitHub 账号下。

2. **在你的 Fork 中开发**  
   - 克隆到本地：`git clone https://github.com/你的用户名/training_camp_2025.git`  
   - 在自己的目录下操作（`方向/姓名/`）  
   - 添加你的代码、README、运行说明等  
   - 提交并推送：`git add .` → `git commit` → `git push`

3. **发起 Pull Request**  
   - 回到 GitHub，在你的 Fork 仓库页面点击 **`Contribute → Open Pull Request`**  
   - 确保目标分支是 `牧码南山:main`  
   - 填写清晰的 PR 标题（如：`[add] - 手写数字识别练习`）  
   - 提交！等待审核合并 ✅

### Q&A

**Q：我没有发现属于我名称的文件夹 or 我也想成为仓库的贡献者**  
**A：欢迎加入！🎉**  

请按以下步骤申请你的专属目录：

1. **[点击这里新建 Issue](https://github.com/CodeDriverTech/training_camp_code_driver/issues/new)**  
2. 在 **Issue 正文** 中按格式填写（示例）：
   ```text
   添加成员 人工智能: ZhangSan
   ```
   > 支持的方向：`人工智能`、`硬件开发`、`网络安全`、`软件开发` 
   ![guide](/images/issue-guide.png) 
3. 提交后，管理员审核通过（打上 `approved` 标签），系统将**自动为你创建个人目录** 

> 提示：无需 Fork 或手动创建文件夹，一切自动化完成
